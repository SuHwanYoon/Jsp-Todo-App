# JSP 기반 할일 어플리케이션


### 1.목표
●JSP기반의 어플리케이션을 만들어 포트폴리오로 활용




### 2.기능
●SpringSecurity를 이용해 로그인,로그아웃


●계정별로 할일,날짜,달성여부등록 및 변경,삭제 가능


●PK인 게시물 번호는 AutoIncrement되며 화면에 표시되지않음 




### 3.개발환경
●WebFrameWork
 -SpringBoot 3.1.5


●FrontEnd
 -BootStrap & JSP

●DB
 -MySQL


●ORM
 -Spring-Data-JPA


●Container 
 -Docker
